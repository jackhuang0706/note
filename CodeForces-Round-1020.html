A. Dr. TC

Statement

有一長度為 n 的 01字串 s 以及一個長度為 n 的字串型態陣列 a1​,a2​,…,an​，其中 ai​ 代表 s 中的第 i 個元素變成 1−si​（0 變成 1，1 變成 0），其他則和 s 相同，請問 a1​ ~ an​ 中所有的 "1" 共有多少個？

  • 1≤n≤10

Solution

因為每一個元素都只會被改變一次，也就是對於每個 si​(1≤i≤n)，si​ 本身會出現 n−1 次，而 1−si​ 會出現一次。透過這一點，可以從頭掃一次 s，若 si​=1 就將答案加上 n−1，反之則將答案 +1。

#include <bits/stdc++.h>
#define fastio ios::sync_with_stdio(0),cin.tie(0),cout.tie(0);
using namespace std;
 
int main(){
    fastio;
    int t;
    cin >> t;
    while(t--){
        int n;
        cin >> n;
        string s;
        cin >> s;
        int ans=0;
        for(int i=0;i<n;i++){
            if(s[i]=='1'){
                ans+=(n-1);
            }
            else{
                ans+=1;
            }
        }
        cout << ans << "\n";
    }
}

B. St. Chroma

Statement

假設有一個長度為 n 的 排列 p（定義為 0 到 n−1 的任意一種可能排列方式） 以及一個長度為 n 的陣列 A。對於每一個 1≤i≤n，Ai​ 會被給予一個編號，而這個編號是 MEX(p1​,p2​,…,pi​)，也就是不在 p1​,p2​,…,pi​ 之中的最小非負整數。舉例來說，若 p=[1,0,3,2]，則 A 會是 [0,2,2,4]。現在給定兩個整數 n,x，請構造出一的排列 p，使得對應的 A 中有最多的 x。

  • 1≤n≤2×105
  • 0≤x≤n

Solution

若要盡量讓 x 存在於 A 中，則必須讓 0 到 x−1 愈早出現愈好。因此先依序將 0,1,…,x−1 加入 p，然後再把 n−1,n−2,…,x+1 加入 p 即可，但 n=x 的時候要另外寫一個特例出來，p=[0,1,2,3,…,n−1]。

#include <bits/stdc++.h>
#define fastio ios::sync_with_stdio(0),cin.tie(0),cout.tie(0);
#define eb emplace_back
using namespace std;
 
int main(){
    fastio;
    int t;
    cin >> t;
    while(t--){
        int n,x;
        cin >> n >> x;
        vector<int> ans;
        if(n==x){
            for(int i=0;i<n;i++){
                cout << i << " ";
            }
            cout << "\n";
            continue;
        }
        for(int i=0;i<n;i++){
            if(i==x){
                break;
            }
            ans.eb(i);
        }
        for(int i=n-1;i>=0;i--){
            if(i==x){
                break;
            }
            ans.eb(i);
        }
        ans.eb(x);
        for(int &i:ans) cout << i << " ";
        cout << "\n";
    }
}
