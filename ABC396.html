A - Triple Four

Statement

給定一個長度為 N 的序列 A=(A1​,A2​,A3​,…,AN​)，請判斷是否存在一個位置 i (1≤i≤N−2) 使得 Ai​=Ai+1​=Ai+2​。

  • 3≤N≤100
  • 1≤Ai​≤100

Solution

只需要用一個迴圈從頭到尾掃過一次，每次用 if 寫個判斷式就會過了。

#include <bits/stdc++.h>
#define fastio ios::sync_with_stdio(0),cin.tie(0),cout.tie(0)
using namespace std;

int main(){
    fastio;
    int n;
    cin >> n;
    vector<int> v(n);
    bool ans=0;
    for(int &i:v) cin >> i;
    for(int i=0;i<n-2;i++){
        if(v[i]==v[i+1] && v[i+1]==v[i+2]){
            ans=1;
        }
    }
    cout << ((ans)?"Yes":"No") << "\n";
}

B - Card Pile

Statement

有 100 張編號為 0、疊成一疊的卡片，現在有 Q 筆操作，每次的操作有兩種：

  1. 把一張編號為 x 的卡片放在這疊卡片的最上面
  2. 輸出最上面的卡片的編號，並且將其移除。另外題目保證這疊卡片必定有至少一張卡片
  • 1≤Q≤100
  • 1≤x≤100

Solution

用一個堆疊模擬過程就結束了。

#include <bits/stdc++.h>
#define fastio ios::sync_with_stdio(0),cin.tie(0),cout.tie(0)
using namespace std;

int main(){
    fastio;
    int q,type,x;
    cin >> q;
    stack<int> s;
    for(int i=0;i<100;i++) s.push(0);
    for(int i=0;i<q;i++){
        cin >> type;
        if(type==1){
            cin >> x;
            s.push(x);
        }
        else{
            cout << s.top() << "\n";
            s.pop();
        }
    }
}

C - Buy Balls

Statement

有 n 個黑球與 m 個白球，每個球都有一個價值，其中這 n 個黑球的價值分別是 B1​,B2​,…,Bn​，而 m 個白球的價值分別是 W1​,W2​,…,Wm​，現在請選擇一些黑球與白球（也可以不選），在 拿到的黑球數量 ≥ 拿到的白球數量 的條件下，使它們價值的總和最大。

  • 1≤n,m≤2×105
  • −109≤Bi​,Wj​≤109

Solution

因為要盡量取價值大的球，先把 B 和 W 由大至小排序，讓價值大的球排在前面，接著可以用迴圈從 1 到 min(n,m) 掃過去，可以發現有以下四種情況：

  • Bi​>0 且 Wi​>0
  • Bi​<0 且 Wi​<0
  • Bi​>0 且 Wi​<0
  • Bi​<0 且 Wi​>0

在第一種情況的話就兩個都拿，因為拿了不可能讓總價值更小。

在第二種情況下則是都不拿並跳出迴圈，因為如果在這個位置兩個價值都 <0 的話，後面的價值只會更小（拿了會使總價值更小）。

在第三種情況，因為沒有規定黑球只能拿多少個，因此把價值 >0 的黑球拿進來可以讓總價值更大。反之，白球不拿。

最後一種情況下，因為要拿就只能兩個都拿，或是兩個都不拿（題目規定白球數量不能大於黑球數量），因此判斷的標準是把這兩個數值相加，若 >0 代表兩個都拿對於總價值增加是有幫助的，因此都拿，反之亦然。

另外在 n>m 的狀況下還需要考慮 Bm+1​,Bm+2​,…,Bn​ 這些黑球能不能拿，有可能到這裡黑球的價值是 >0 的。（比賽的時候被卡了好幾次...）

#include <bits/stdc++.h>
#define fastio ios::sync_with_stdio(0),cin.tie(0),cout.tie(0)
#define int long long
using namespace std;

signed main(){
    fastio;
    int n,m;
    cin >> n >> m;
    vector<int> b(n),w(m);
    for(int &i:b) cin >> i;
    for(int &i:w) cin >> i;
    sort(b.begin(),b.end(),greater<int>());
    sort(w.begin(),w.end(),greater<int>());
    int ans=0,v=min(n,m);
    for(int i=0;i<v;i++){
        if(b[i]>0 && w[i]>0){
            ans+=b[i]+w[i];
        }
        else if(b[i]>0 && w[i]<0){
            ans+=b[i];
        }
        else if(b[i]<0 && w[i]>0){
            if(b[i]+w[i]>0) ans+=b[i]+w[i];
        }
        else{
            break;
        }
    }
    if(n>m){
        for(int i=v;i<n;i++){
            if(b[i]>0){
                ans+=b[i];
            }
            else{
                break;
            }
        }
    }
    cout << ans << "\n";
}
