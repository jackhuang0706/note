前綴和、差分與離散化

前綴和

前綴和指的是在一陣列 a 中前 k 項的總和，通常會使用一個陣列來表示。例如，有一陣列 a1​,a2​,a3​,…,an​ ，並定義前綴和陣列為 pre，則根據定義可寫成： prek​=a1​+a2​+a3​+⋯+ak​=i=1∑k​ai​ ( ∀ 1≤k≤n ) 一般來說，前綴和的使用時機通常在大量詢問陣列區間和的時候。一個經典的問題：

有一長度為 n 的陣列 a1​,a2​,a3​,…,an​ 及 q 筆詢問，每個詢問包含 2 個整數 l,r ，對於每個 l,r ，請輸出 al​+al+1​+al+2​+⋯+ar−1​+ar​ 的值。

  • 1≤n,q≤2×105
  • −109≤ai​≤109
  • 1≤l≤r≤n

如果對於每次詢問都用陣列從 l 到 r 遍歷，複雜度會是 O(nq)，會吃 TLE。注意到其實每次詢問可能存在大量重疊的區間，因此可以先用前綴和預處理，把每個 a1​ 到 ak​ 的值先算出來，然後對於每個 [l,r] 只要輸出 prer​−prel−1​ 就好，這樣預處理的複雜度是 O(n)，而每次詢問的複雜度則是 O(1)，總複雜度只有 O(n+q)，比暴力好多了。

離散化

歐幾里得演算法（輾轉相除法）的證明

設有兩數 a,b (a,b∈N, a>b)，且 a,b 的最大公因數為 g，則在遞迴時的過程可以用下式表示：

a=q0​b+r0​ (0≤r0​<b)b=q1​r0​+r1​ (0≤r1​<r0​)r0​=q2​r1​+r2​ (0≤r2​<r1​)…rk−2​=qk​rk−1​+rk​ (0≤rk​<rk−1​)

透過以上式子，可知 0≤rk​<rk−1​<rk−2​<⋯<r1​<r0​<b<a。因為 rk​ 是不斷取餘數所產生的結果，因此必然存在 n 使得 rn​=0（因為 rk​ 只會愈來愈小，並且保持 >0）。

證明 1

因 rn​=0，故 rn−2​=qn​rn−1​，因此 rn−1​ 整除 rn−2​，即 rn−1​∣rn−2​。又

rn−3​=qn−1​rn−2​+rn−1​=qn−1​qn​rn−1​+rn−1​=rn−1​ (qn−1​qn​+1)⇒rn−1​∣rn−3​

類似的方法可推到 rn−4​,rn−5​,…,r2​,r1​,b,a，因此可確認 rn−1​∣b 且 rn−1​∣a，也就是 rn−1​ 為 a,b 的公因數，也因此確認 rn−1​≤g（任何公因數必不大於 gcd）

證明 2

因 g 為 a,b 的公因數，故可將 a 令為 mg，b 令為 ng (m,n∈Z+,m>n)，套入 a=q0​b+r0​ 可得

mg=q0​ng+r0​⇒r0​=mg−q0​ng=g(m−q0​n)

將 r0​ 再套入 b=q1​r0​+r1​ 為

ng=q1​g(m−q0​n)+r1​⇒r1​=g(n−q1​(m−q0​n))

同理可知所有的 rk​ 皆為 g 的整數倍， rn−1​ 也是，因此可得到 rn−1​≥g。

結論

根據證明 1,2 得到下列式子：

{rn−1​≤grn−1​≥g​⇒rn−1​=g

可證明輾轉相除法的方法最後可得到正確的 gcd。

貝祖定理（Bézout's lemma）證明

貝祖定理： 若一線性丟番圖方程式 ax+by=m 有 (x,y) 整數解，則 m 必為 gcd(a,b) 的倍數，也就是 gcd(a,b)∣m。

證明

先將方程式換成 a>0,b>0,m>0 的寫法（若原式的 m<0，將整個方程式加上負號，若 a,b<0，令 x′=−x 或 y′=−y，再用 ax′+by′=m 下去算，算完再代換），假設 ax+by=m 的所有解可以用一個集合 A 來表示，即 A={ax+by∣(x,y)∈Z2}，由於 A 中所有元素必 >0（定義 m>0），因此必有最小值。現在假設此最小值為 m0​，且 ax0​+by0​=m0​。

假設 A 中有另一元素 m1​，滿足 ax1​+by1​=m1​，考慮 m1​ 除以 m0​ 後的結果，為：

m1​=qm0​+r(0≤r<m0​)

代換後可寫成：

ax1​+by1​=q(ax0​+by0​)+r⇒r=a(x1​−qx0​)+b(y1​−qy0​)

注意到這個形式是符合 ax+by 的，也就是 r 是 A 中的元素，這與上面假設的 m0​ 為 A 中的最小元素不符（出現一個更小的元素 r），因此 r=0（A 中不包含 0 這個元素）。從這個結論，可知道

m1​=qm0​

也就是 A 中任一元素必為 m0​ 的倍數。若套用到 a,b 上面（a,b 也是 A 中的元素，套用 (x,y)=(1,0) 和 (0,1) 可知）就是 m0​∣a 且 m0​∣b，故 m0​ 是 a,b 的公因數。

考慮到任意一個 a,b 的公因數 d，令 a=kd,b=ld，則代換以下式子：

ax0​+by0​=m0​=(kd)x0​+(ld)y0​=m0​⇒d(kx0​+ly0​)=m0​⇒d∣m0​

因此 m0​≥ 任一個 a,b 的公因數，同時也確認了 m0​ 是 a,b 的 gcd。

A. Dr. TC

Statement

有一長度為 n 的 01字串 s 以及一個長度為 n 的字串型態陣列 a1​,a2​,…,an​，其中 ai​ 代表 s 中的第 i 個元素變成 1−si​（0 變成 1，1 變成 0），其他則和 s 相同，請問 a1​ ~ an​ 中所有的 "1" 共有多少個？

  • 1≤n≤10

Solution

因為每一個元素都只會被改變一次，也就是對於每個 si​(1≤i≤n)，si​ 本身會出現 n−1 次，而 1−si​ 會出現一次。透過這一點，可以從頭掃一次 s，若 si​=1 就將答案加上 n−1，反之則將答案 +1。

#include <bits/stdc++.h>
#define fastio ios::sync_with_stdio(0),cin.tie(0),cout.tie(0);
using namespace std;
 
int main(){
    fastio;
    int t;
    cin >> t;
    while(t--){
        int n;
        cin >> n;
        string s;
        cin >> s;
        int ans=0;
        for(int i=0;i<n;i++){
            if(s[i]=='1'){
                ans+=(n-1);
            }
            else{
                ans+=1;
            }
        }
        cout << ans << "\n";
    }
}

B. St. Chroma

Statement

假設有一個長度為 n 的 排列 p（定義為 0 到 n−1 的任意一種可能排列方式） 以及一個長度為 n 的陣列 A。對於每一個 1≤i≤n，Ai​ 會被給予一個編號，而這個編號是 MEX(p1​,p2​,…,pi​)，也就是不在 p1​,p2​,…,pi​ 之中的最小非負整數。舉例來說，若 p=[1,0,3,2]，則 A 會是 [0,2,2,4]。現在給定兩個整數 n,x，請構造出一的排列 p，使得對應的 A 中有最多的 x。

  • 1≤n≤2×105
  • 0≤x≤n

Solution

若要盡量讓 x 存在於 A 中，則必須讓 0 到 x−1 愈早出現愈好。因此先依序將 0,1,…,x−1 加入 p，然後再把 n−1,n−2,…,x+1 加入 p 即可，但 n=x 的時候要另外寫一個特例出來，p=[0,1,2,3,…,n−1]。

#include <bits/stdc++.h>
#define fastio ios::sync_with_stdio(0),cin.tie(0),cout.tie(0);
#define eb emplace_back
using namespace std;
 
int main(){
    fastio;
    int t;
    cin >> t;
    while(t--){
        int n,x;
        cin >> n >> x;
        vector<int> ans;
        if(n==x){
            for(int i=0;i<n;i++){
                cout << i << " ";
            }
            cout << "\n";
            continue;
        }
        for(int i=0;i<n;i++){
            if(i==x){
                break;
            }
            ans.eb(i);
        }
        for(int i=n-1;i>=0;i--){
            if(i==x){
                break;
            }
            ans.eb(i);
        }
        ans.eb(x);
        for(int &i:ans) cout << i << " ";
        cout << "\n";
    }
}

A - Triple Four

Statement

給定一個長度為 N 的序列 A=(A1​,A2​,A3​,…,AN​)，請判斷是否存在一個位置 i (1≤i≤N−2) 使得 Ai​=Ai+1​=Ai+2​。

  • 3≤N≤100
  • 1≤Ai​≤100

Solution

只需要用一個迴圈從頭到尾掃過一次，每次用 if 寫個判斷式就會過了。

#include <bits/stdc++.h>
#define fastio ios::sync_with_stdio(0),cin.tie(0),cout.tie(0)
using namespace std;

int main(){
    fastio;
    int n;
    cin >> n;
    vector<int> v(n);
    bool ans=0;
    for(int &i:v) cin >> i;
    for(int i=0;i<n-2;i++){
        if(v[i]==v[i+1] && v[i+1]==v[i+2]){
            ans=1;
        }
    }
    cout << ((ans)?"Yes":"No") << "\n";
}

B - Card Pile

Statement

有 100 張編號為 0、疊成一疊的卡片，現在有 Q 筆操作，每次的操作有兩種：

  1. 把一張編號為 x 的卡片放在這疊卡片的最上面
  2. 輸出最上面的卡片的編號，並且將其移除。另外題目保證這疊卡片必定有至少一張卡片
  • 1≤Q≤100
  • 1≤x≤100

Solution

用一個堆疊模擬過程就結束了。

#include <bits/stdc++.h>
#define fastio ios::sync_with_stdio(0),cin.tie(0),cout.tie(0)
using namespace std;

int main(){
    fastio;
    int q,type,x;
    cin >> q;
    stack<int> s;
    for(int i=0;i<100;i++) s.push(0);
    for(int i=0;i<q;i++){
        cin >> type;
        if(type==1){
            cin >> x;
            s.push(x);
        }
        else{
            cout << s.top() << "\n";
            s.pop();
        }
    }
}

C - Buy Balls

Statement

有 n 個黑球與 m 個白球，每個球都有一個價值，其中這 n 個黑球的價值分別是 B1​,B2​,…,Bn​，而 m 個白球的價值分別是 W1​,W2​,…,Wm​，現在請選擇一些黑球與白球（也可以不選），在 拿到的黑球數量 ≥ 拿到的白球數量 的條件下，使它們價值的總和最大。

  • 1≤n,m≤2×105
  • −109≤Bi​,Wj​≤109

Solution

因為要盡量取價值大的球，先把 B 和 W 由大至小排序，讓價值大的球排在前面，接著可以用迴圈從 1 到 min(n,m) 掃過去，可以發現有以下四種情況：

  • Bi​>0 且 Wi​>0
  • Bi​<0 且 Wi​<0
  • Bi​>0 且 Wi​<0
  • Bi​<0 且 Wi​>0

在第一種情況的話就兩個都拿，因為拿了不可能讓總價值更小。

在第二種情況下則是都不拿並跳出迴圈，因為如果在這個位置兩個價值都 <0 的話，後面的價值只會更小（拿了會使總價值更小）。

在第三種情況，因為沒有規定黑球只能拿多少個，因此把價值 >0 的黑球拿進來可以讓總價值更大。反之，白球不拿。

最後一種情況下，因為要拿就只能兩個都拿，或是兩個都不拿（題目規定白球數量不能大於黑球數量），因此判斷的標準是把這兩個數值相加，若 >0 代表兩個都拿對於總價值增加是有幫助的，因此都拿，反之亦然。

另外在 n>m 的狀況下還需要考慮 Bm+1​,Bm+2​,…,Bn​ 這些黑球能不能拿，有可能到這裡黑球的價值是 >0 的。（比賽的時候被卡了好幾次...）

#include <bits/stdc++.h>
#define fastio ios::sync_with_stdio(0),cin.tie(0),cout.tie(0)
#define int long long
using namespace std;

signed main(){
    fastio;
    int n,m;
    cin >> n >> m;
    vector<int> b(n),w(m);
    for(int &i:b) cin >> i;
    for(int &i:w) cin >> i;
    sort(b.begin(),b.end(),greater<int>());
    sort(w.begin(),w.end(),greater<int>());
    int ans=0,v=min(n,m);
    for(int i=0;i<v;i++){
        if(b[i]>0 && w[i]>0){
            ans+=b[i]+w[i];
        }
        else if(b[i]>0 && w[i]<0){
            ans+=b[i];
        }
        else if(b[i]<0 && w[i]>0){
            if(b[i]+w[i]>0) ans+=b[i]+w[i];
        }
        else{
            break;
        }
    }
    if(n>m){
        for(int i=v;i<n;i++){
            if(b[i]>0){
                ans+=b[i];
            }
            else{
                break;
            }
        }
    }
    cout << ans << "\n";
}

P-1-7. 子集合乘積

#include <bits/stdc++.h>
#define fastio ios::sync_with_stdio(0),cin.tie(0),cout.tie(0);
#define int long long
#define mod 10009
using namespace std;

const int maxN=30;
vector<int> arr(maxN);
int n,ans=0;

void solve(int cnt,int total){
    if(cnt>=n){
        if(total%mod==1) ans++;
        return;
    }
    solve(cnt+1,(total*arr[cnt])%mod);
    solve(cnt+1,total);
    return;
}
signed main(){
    fastio;
    cin >> n;
    for(int i=0;i<n;i++) cin >> arr[i];
    solve(0,1);
    cout << ans-1 << "\n";
}

Q-1-8. 子集合的和

#include <bits/stdc++.h>
#define fastio ios::sync_with_stdio(0),cin.tie(0),cout.tie(0);
#define int long long
using namespace std;

const int maxN=30;
vector<int> arr(maxN);
int n,p,ans=0;

void solve(int cnt,int total){
    if(cnt>=n){
        if(total<=p){
            ans=max(ans,total);
        }
        return;
    }
    solve(cnt+1,total+arr[cnt]);
    solve(cnt+1,total);
    return;
}
signed main(){
    fastio;
    cin >> n >> p;
    for(int i=0;i<n;i++) cin >> arr[i];
    solve(0,0);
    cout << ans << "\n";
}

Q-2-5. 快速計算費式數列第 n 項

#include <bits/stdc++.h>
#define fastio ios::sync_with_stdio(0),cin.tie(0),cout.tie(0);
#define int long long
#define mod 1000000007
using namespace std;

vector<vector<int>> times(vector<vector<int>> a,vector<vector<int>> b){
    vector<vector<int>> result(2,vector<int> (2));
    result[0][0]=a[0][0]*b[0][0]+a[0][1]*b[1][0];result[0][0]%=mod;
    result[0][1]=a[0][0]*b[0][1]+a[0][1]*b[1][1];result[0][1]%=mod;
    result[1][0]=a[1][0]*b[0][0]+a[1][1]*b[1][0];result[1][0]%=mod;
    result[1][1]=a[1][0]*b[0][1]+a[1][1]*b[1][1];result[1][1]%=mod;
    return result;
}
vector<vector<int>> cal(vector<vector<int>> arr,int p){
    vector<vector<int>> result={{1,0},{0,1}};
    while(p>0){
        if(p&1){
            result=times(result,arr);
        }
        arr=times(arr,arr);
        p>>=1;
    }
    return result;
}
signed main(){
    fastio;
    int n;
    while(cin >> n){
        if(n==-1){
            break;
        }
        vector<vector<int>> arr={{1,1},{1,0}};
        vector<vector<int>> ans=cal(arr,n-1);
        ans[0][0]%=mod;
        cout << ans[0][0] << "\n";
    }
}
