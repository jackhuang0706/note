mdbook操作

註：以下命令皆運行於 Windows Powershell 的環境

  • mdbook init <書名>：建構一本書，另外書名要加 .md（在當前目錄下執行）。
  • mdbook serve：在 localhost 上試跑，端口為 :3000，到書籍的根目錄下執行（打 ls 會出現 book、src 資料夾那裡）。
  • New-Item <文件名>：新增文件（在當前目錄下執行）
  • Remove-Item <文件名>：刪除文件（在當前目錄下執行）
  • Rename-Item <文件名>：重新命名文件

將內容推送到 github 上

  1. 到書籍根目錄下
  2. mdbook build
  3. cd book
  4. git init
  5. git remote add origin https://github.com/jackhuang0706/note.git
  6. git checkout -b gh-pages
  7. git add .
  8. git commit -m "Deploy"
  9. git push origin gh-pages --force
  10. 網站在https://jackhuang0706.github.io/note/

把以上內容寫成腳本（Windows Powershell）

把以上這麼多步驟寫成一個腳本，在修改內容及更新網站時就不用再一個一個步驟重打了。

  1. 建立一個檔案 deploy.ps1（書籍根目錄）
  2. 加入內容：
# Build the mdBook
mdbook build

# Change directory to book
cd .\book

# Initialize a git repository (if not already initialized)
git init

# Add remote origin (replace with your own GitHub repository)
git remote add origin https://github.com/jackhuang0706/note.git

# Switch to gh-pages branch
git checkout -b gh-pages

# Add changes, commit, and push to GitHub
git add .
git commit -m "Deploy update"
git push origin gh-pages --force

# Go back to the root folder
cd ..
  1. 更新時輸入 .\deploy.ps1 就完成了

[output.html]
